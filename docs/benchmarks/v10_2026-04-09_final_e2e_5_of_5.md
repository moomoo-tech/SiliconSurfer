## Key Architectural Features That Won

1. **see_page()** — Distill Playwright's actual page content (with cookies/session).
   Without this, Agent couldn't see it had already logged in.

2. **Multi-mode vision** — Spider for links, Operator for forms, Reader for content.
   Agent auto-selects the best lens per step.

3. **@e element references** — `@e3 [Input: name=username]` → Agent says "fill @e3".
   No CSS selector guessing.

4. **State analysis prompt** — Forces LLM to analyze before acting.
   Prevents the "State Blindness" loop problem.

5. **Loop detection** — Warns LLM when it's revisiting same URLs.
   Forces "done" with collected data.
