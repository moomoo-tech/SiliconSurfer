## 6. Bugs Fixed (during v0 development)

| Bug | Root Cause | Fix |
|-----|-----------|-----|
| Python.org garbled | reqwest missing gzip/brotli | Added feature flags |
| HN table bloat 14K→6K | Layout table rendered as md table | `is_data_table()` heuristic |
| Nav repeated 3x | No line-level dedup | `dedup_and_clean()` |
| MDN content gone (0 chars) | `[class*='banner']` killed layout container | Narrowed noise selector scope |
| lol_html `&#160;` not decoded | No entity decoding | `decode_entities()` |
| `<pre>` indentation collapsed | Whitespace collapsing inside pre | `[[PRE_START/END]]` state markers |
| Relative links lost | Only matched `http` prefix | base URL resolution |
| `<title>` leaked into body | lol_html didn't remove `<head>` | Added `head` to noise selectors |
